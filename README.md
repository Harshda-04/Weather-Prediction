 🌦️ Weather Prediction AI (Machine Learning Project)

An end-to-end Machine Learning project that predicts weather conditions using historical meteorological and time-based data. The system is built using **Random Forest Regressor** and deployed using **Streamlit** for an interactive user experience.

## 🧠 Project Overview
This project uses machine learning to predict weather-related values based on environmental and temporal features. It demonstrates a complete ML pipeline including:

- Data preprocessing  
- Feature selection  
- Model training  
- Evaluation (R² Score ~0.94)  
- Deployment using Streamlit  

---

## 📊 Features Used for Prediction
- 💧 Humidity (%)  
- 🌬️ Wind Speed  
- 📊 Mean Pressure  
- 📅 Year  
- 📅 Month  
- 📅 Day  

---

## 🤖 Machine Learning Model
- Algorithm: Random Forest Regressor  
- Type: Supervised Regression  
- Performance: R² Score = 0.94  
- Output: Continuous weather value (e.g., temperature prediction)

---

## 🖥️ Application Features
- Interactive Streamlit dashboard  
- Real-time prediction system  
- Clean and responsive UI  
- Weather interpretation (Hot / Normal / Cold)  
- Error-safe model loading
